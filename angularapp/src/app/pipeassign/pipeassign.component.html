Built in pipe & Custom pipe examples

Date pipe :   This pipe is used for the transformation of dates.
  • Today's Date : {{todayDate}}
  • Date : {{todayDate | date}}
  • Short Date & Time : {{todayDate | date:'short'}}
  • Full Date : {{todayDate | date:'fullDate'}}
  • Lowercase Format : {{todayDate | date:'fullDate' | lowercase}}
  • Uppercase Format : {{todayDate | date:'fullDate' | uppercase}}
Currency Pipe:   This pipe is used for formatting currencies. -
  • Price : {{price}}
  • Currency price in India : {{price | currency: 'INR'}}
  • Currency price in USD : {{price | currency: 'USD'}}
  • Currency price in European : {{price | currency :'EUR' }}
Json Pipe :   This transforms a JavaScript object into a JSON string.
  • Json Object Format: {{obj | json}}
  • Object in Lowercase format: {{obj | json | lowercase}}
async :   If data comes in the form of observables, then Async pipe subscribes to an observable and returns the transmitted values
  • get value: {{observable | async}}

Custom pipe:

  • string : {{val}}
  • string length : {{val | count}}
